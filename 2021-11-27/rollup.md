# Rollup 筆記

## 什麼事 Rollup ?

rollup 是一個編譯的工具 , 專門集中在 esm 的打包 , 因此速度可以比 babel 還高

## Rollup 的初步探索

將 rollup 安裝在電腦中

```shell
~$ npm install --global rollup
```

起始專案

- [rollup-starter-lib](https://github.com/rollup/rollup-starter-lib)
- [rollup-starter-app](https://github.com/rollup/rollup-starter-app)

## 那如何使用 Rollup 呢 ?

- CLI 模式
- API 模式
- 與 Gulp 工具結合使用
